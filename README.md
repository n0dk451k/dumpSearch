# dumpSearch
поиск сигнатур в бинарных дампах 

Использование:
  ./search <file> [options]
```
Поиск:
  --utf8 <string>
  --utf16le <string>
  --utf16be <string>
  --hex <hexstring>

Вывод:
  --xxd "<args>"
  --strings "<args>"
  --strings-len <N>
  --offset-add <N>
  --progress <seconds>
  --help
```
